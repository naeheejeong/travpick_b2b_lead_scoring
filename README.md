# TravPick 사용자 행동 측정 체계 설계

> RavenL 콘텐츠 사이언티스트 인턴 · 2025.11 ~ 2025.12 · 단독 수행(100%)

측정 체계가 없던 앱에서 콘텐츠 유입부터 구매 전환까지 추적 가능한 사용자 행동 분석 환경을 0→1로 구축한 프로젝트

## 🎯 배경 · 문제정의
- MVP 출시 후 Product Hunt 글로벌 런칭 준비 중이었으나, 홍보 채널은 인스타그램 1개뿐
- 채널을 확장해도 어떤 채널·콘텐츠가 성과를 내는지 판단할 KPI 기준이 없었음
- 진짜 문제는 데이터 부족이 아니라 **무엇을 측정할지 정하는 KPI 체계의 부재**

## 🗂 데이터 · 방법
- LinkedIn, Medium, Reddit, Note, Ameba, X, 네이버 블로그 등 채널 확장 및 콘텐츠 발행
- UTM 파라미터 설계·표준화 → GA4·GTM 기반 측정 인프라 구축 (Firebase 연동, CTA 클릭 추적)
- 퍼널 분석(이탈구간 식별) · 채널별 유입 분석 대시보드 제작
- A/B 테스트: 공감형 vs 정보전달형 콘텐츠 반응 비교 (Reddit 1.3K+ views)
