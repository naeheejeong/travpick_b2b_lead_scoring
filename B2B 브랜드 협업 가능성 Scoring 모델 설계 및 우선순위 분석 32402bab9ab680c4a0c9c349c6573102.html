B2B 브랜드 협업 가능성 Scoring 모델 설계 및 우선순위 분석

문제 정의

TravPick 서비스 확장을 위해 미국/한국의 뷰티/푸드 브랜드와 B2B 협업을 진행하고자 하였지만 문제가 발생함

  • 너무 많은 후보 브랜드
  • 어떤 브랜드에 먼저 컨택해야할지 기준 없음
  • 리드 우선순위 없음

따라서 다음과 같이 질문을 던져 해결하고자 함.

어떤 브랜드가 TravPick과 협업할 가능성이 높은가?
(브랜드 컨택 전략 개선방안 제시)

데이터 구축

브랜드 데이터 수집

미국 / 한국 뷰티브랜드 약 200개 이상 수집

변수설명추가 분류
brand브랜드 이름
country국가
scale회사 규모Enterprise / Mid-size / Growth DTC / Indie
subcategory_1제품 카테고리

모델 설계 방식

학습된 데이터/기존 데이터가 존재하지 않고 직접 만들었기때문에 → 지도학습 대신 Rule-Based Scoring Model 을 설계함.

스코어링 규칙

브랜드 특성에 따라 점수를 부여

조건점수
scale == Mid-size+2
scale == Growth DTC+1
subcategory == Hair tools+1
country == US+1

스코어 계산 방식

score = 0

if scale == Mid-size:
    score += 2

if scale == Growth DTC:
    score += 1

if subcategory == Hair tools:
    score += 1

if country == US:
    score += 1

모델의 의미

이 모델은 다음 역할을 수행합니다.

기존 방식

무작위 브랜드 컨택

개선 방식

데이터 기반 리드 우선순위

Score 3~4  → High Priority
Score 2    → Medium
Score 0~1  → Low Priority

향후 발전 방향 (머신러닝)

현재 모델은 Rule-based

하지만 데이터가 쌓이면 → 다음 ML 모델로 확장 가능할 것

ex-

  • Logistic Regression
  • Random Forest
  • XGBoost

목표 : P (브랜드가 협업에 응당할 확률)
